Runway Compiler

C++

Runway is an approach to create a small script-like programming language with native-code output. The project includes the compiler frontend which contains the tokenizer, parser and code-generator. The output is LLVM-IR (bytecode). The final conversion to the target machine code instructions will be done by the LLVM static compiler LLVM static compiler.

Runway LLVM IR output example

                
;declare c puts function
declare  i32 @puts(i8*) 

;create global string constant (12 chars with one byte each)
@global_str = constant [12 x i8] c"joshtastic\00"

;define entry point
define i32 @main { 

  ;call the puts function with the string-constant
  call i32 @puts( [12 x i8] @global_str )
  
  ;return exit code 0
  ret i32 0 
}

Generic CSV Writer

Java

CSV-Writer is a tiny library that helps to export CSV-Data. It's designed to be highly extensible by the user to create custom transformers.

            

                
class Bean {

    @CsvEntry(0)
    String name;

    @CsvEntry(1)
    String homeCountry;

    @CsvEntry(2)
    float price;
}
